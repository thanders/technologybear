Cloud architecture design principles are the final part of the AWS CCP Cloud Concepts. The principles are imporant to consider when designing a cloud system.


## Explain the design principles
- Design for failure
- Decouple components versus monolithic architecture
- Implement elasticity in the cloud versus on-premises
- Think parallel
